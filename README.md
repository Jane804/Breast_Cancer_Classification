# Breast Cancer Classification using Supervised Learning

## Objective
This project evaluates five supervised learning algorithms on the sklearn breast cancer dataset.

## Dataset
Breast Cancer Wisconsin Dataset from sklearn.

## Algorithms Implemented
1. Logistic Regression
2. Decision Tree
3. Random Forest
4. Support Vector Machine
5. k-Nearest Neighbors

## Preprocessing
- Checked for missing values (none found)
- Applied StandardScaler for feature scaling
- Train-test split (80-20)

## Results
Random Forest and SVM performed the best with highest accuracy.
Decision Tree performed the lowest due to overfitting tendency.
